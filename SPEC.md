# YouTubeToMP3 Bot — Спецификация

## Описание
Telegram-бот для скачивания аудио из YouTube видео и поиска видео по названию.

## Milestones

### v0.1.0 — MVP: Скачивание аудио
- [x] Скачивание аудио из YouTube по ссылке (yt-dlp)
- [x] Проверка длительности видео (макс. 30 мин)
- [x] Проверка размера файла (макс. 50 MB)
- [x] Автоматическая очистка файлов
- [x] Docker / docker-compose конфигурация
- [x] Команда `/start`

### v0.2.0 — Статистика и админка
- [x] SQLite база для отслеживания пользователей и загрузок
- [x] Команда `/stats` (только для админов)
- [x] Уведомления админам о новых пользователях
- [x] Семафор на одновременные загрузки
- [x] Очистка URL от параметров плейлиста
- [x] Volume для сохранения БД в Docker

### v0.3.0 — Поиск и рассылка
- [x] Поиск видео через YouTube Data API v3
- [x] Fallback на yt-dlp при ошибке API
- [x] Пагинация результатов (30 видео, по 10 на страницу)
- [x] Рассылка `/broadcast` через FSM (текст, фото, видео)
- [x] Раздельное меню команд для юзеров и админов

### v0.4.0 — Распознавание трека по голосовому
- [x] Приём голосовых сообщений от пользователя
- [x] Распознавание трека через Shazam API (shazamio)
- [x] Поиск распознанного трека в YouTube (название + исполнитель)
- [x] Автоматический выбор первого результата поиска
- [x] Скачивание аудио и отправка в чат (переиспользуем существующий пайплайн)
- [x] Обработка ошибок: трек не распознан / не найден на YouTube
- [x] Лимит длительности голосового сообщения (до 30 секунд)

### v0.5.0 — Улучшения UX (планируется)

- [ ] История загрузок пользователя `/history`
- [ ] Повторная загрузка из истории
- [ ] Выбор формата: MP3 / M4A / OGG
